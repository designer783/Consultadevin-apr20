# Consulta VIN - Figma Make Application

Modern VIN checking service website built with React, TypeScript, Vite, and Tailwind CSS v4 for Figma Make.

## Project Structure

This is a **Figma Make** project using Vite + React (not Next.js):

```
src/
├── app/
│   ├── components/          # 12 Reusable Components
│   │   ├── Navigation.tsx
│   │   ├── Button.tsx
│   │   ├── Badge.tsx
│   │   ├── SectionHeader.tsx
│   │   ├── FeatureCard.tsx
│   │   ├── FeatureWithImage.tsx
│   │   ├── SimpleFeatureCard.tsx
│   │   ├── ServiceCard.tsx
│   │   ├── PricingCard.tsx
│   │   ├── TestimonialCard.tsx
│   │   ├── FAQItem.tsx
│   │   └── ComponentShowcase.tsx
│   └── App.tsx              # Main application component
├── styles/
│   ├── fonts.css            # Google Fonts imports
│   ├── theme.css            # Brand colors & design tokens
│   └── index.css            # Main stylesheet
├── imports/                 # Imported assets

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool & dev server  
- **Tailwind CSS v4** - Utility-first CSS
- **Lucide React** - Icon library

## Brand Guidelines

### Colors

- **Primary Red**: #A51C31 - Energy & Action
- **Navy Blue**: #2C234D - Trust & Stability  
- **Warm Orange**: #F9B248 - Vibrancy & Approachability
- **Light Gray**: #F5F1FE - Neutral Background
- **Dark Gray**: #4A4A4A - Supporting Text

### Typography

- **Headings**: Bricolage Grotesque (200-800 weights)
- **Body**: Rubik (300-800 weights)

## Components

1. **Button** - 4 variants × 3 sizes
2. **Badge** - 3 variants for labels
3. **SectionHeader** - Consistent section titles
4. **FeatureWithImage** - Two-column layouts
5. **SimpleFeatureCard** - Minimal icon cards
6. **PricingCard** - Pricing tiers
7. **TestimonialCard** - Customer reviews
8. **FAQItem** - Accordion FAQ
9. **Navigation** - Responsive header
10. **ComponentShowcase** - Component library viewer

Click "View Components" button (bottom-right) to see all components and variants.
